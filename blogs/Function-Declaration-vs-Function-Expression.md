# Function Declaration vs Function Expression: What’s the Difference?

## Topics to Cover
- What functions are and why we need them
- Function declaration syntax
- Function expression syntax
- Key differences between declaration and expression
- Basic idea of hoisting (very high level)
- When to use each type

Suggestions
- Start by explaining functions as reusable blocks of code
- Show simple examples like adding two numbers
- Compare declaration and expression side by side
- Explain hoisting in simple language (what works before definition and what doesn’t)
- Avoid deep execution context details
- Keep examples small and practical

Assignment Idea
- Write a function declaration that multiplies two numbers
- Write the same logic using function expression
- Call both functions and print results
- Try calling them before defining and observe behavior
