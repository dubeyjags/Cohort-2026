# Express with typescript
- installation
- run commnads 
- ts to js
- server, env, route, controller
- npm i typescript -D
- npm i tsc-watch -D
- npm i express 
- npm i @types/express -D
- npm i @types/node -D

# cmds
- npm init -y
- tsc --init // update rootDir and outDir
- tsc -p . // compile the ts to js
- node dist/index.js // run the js file
- tsc-watch --onSuccess \"node dist/index.js\"
- npm i @types/express -D // for typescript uses
- npm i @types/node -D // for fs and other node realted uses
- npm i zod // for validations
